void Attribute::setDataBits(OpenVisa::DataBits bits)

设置数据位 bits, 仅串口连接时有效.

See also dataBits.

void Attribute::setDeviceName(std::string_view name)

设置设备名称 name, 该设置不是必须的，如果设置了设备名称，抛出异常时， 异常描述将以该设备名称为前缀，用于多台设备控制时的区分，若未设置设备名，则抛出的异常描述无前缀.

See also deviceName.

void Attribute::setFlowControl(OpenVisa::FlowControl fc)

设置流控制 fc, 仅串口连接时有效.

See also flowControl.

void Attribute::setParity(OpenVisa::Parity p)

设置校验规则 p, 仅串口连接时有效.

See also parity.

void Attribute::setStopBits(OpenVisa::StopBits bits)

设置停止位 bits, 仅串口连接时有效.

See also stopBits.

void Attribute::setTerminalChars(std::string_view chars)

设置终结符 chars, 默认为 '\n'(0x0A), 设置这个属性值不会影响它是否被使用.

See also terminalChars.

void Attribute::setTerminalCharsEnable(bool enable)

设置终止符是否启用 enable 默认为启用，该属性仅对读取数据有效.

See also terminalCharsEnable.

void Attribute::setTimeout(const std::chrono::milliseconds &timeout)

设置通信超时时间 timeout.

See also timeout.

std::optional<StopBits> Attribute::stopBits() const
