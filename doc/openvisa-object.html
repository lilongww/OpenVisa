template <typename Args> std::string Object::query(const std::string &fmt, const Args &... args)

格式化发送查询scpi指令，参考 std::format 格式化设置, fmt, args, 并返回查询值.

See also send and readAll.

std::tuple<std::string, bool> Object::read(unsigned long blockSize)

读取 blockSize 大小的数据，返回读到的数据，当没有更多的数据可读时，布尔值为false.

See also readAll.

std::string Object::readAll()

读取所有数据.

See also read.

std::string Object::removeTermChars(const std::string &source) const

移除字符串 source 尾部的终结符.

Note: 若未启用终结符，则该函数不做任何处理.

See also query and readAll.

template <typename Args> void Object::send(const std::string &fmt, const Args &... args)

格式化发送scpi指令，参考 std::format 格式化设置, fmt, args.

See also readAll and read.

[static] template <int T> std::string Object::toVisaAddressString(const int &addr)

将地址类型 addr 转换为 Visa 连接描述字符串.

See also fromVisaAddressString.
