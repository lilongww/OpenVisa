void CommonCommand::cls()

清除事件寄存器位和错误队列.

int CommonCommand::ese()

事件状态查询命令。(*ESE?).

See also setEse.

int CommonCommand::esr()

标准事件状态寄存器查询。 查询标准事件寄存器组的事件寄存器。 事件寄存器是只读寄存器，从条件寄存器锁存事件。 设置事件位时，将忽略与该位相对应的随后发生的事件。(*ESR?).

std::string CommonCommand::idn()

标识查询。 返回仪器的标识字符串.

bool CommonCommand::opc()

在所有的未决命令完成后，将 1 返回到输出缓冲器。 在该命令完成之前，无法执行其他命令。

See also setOpc.

std::vector<std::string> CommonCommand::opt()

返回标识任何已安装选件的字符串。

void CommonCommand::rcl(int i)

调用寄存器保存文件 i.

void CommonCommand::rst()

重置仪表状态.

void CommonCommand::setEse(int en)

事件状态启用命令 en, 为标准事件寄存器组启用使能寄存器中的位。 随后将所选位报告给状态字节寄存器的 5 位。 使能寄存器可定义事件寄存器中的哪些位将被报告给“状态字节”寄存器组。 使能寄存器是可读写的。 (*ESE).

See also ese.

void CommonCommand::setOpc()

在完成当前操作时，在标准事件寄存器中设置“操作完成”（0 位）。(*OPC).

See also opc.

void CommonCommand::setSre(int status)

服务请求启用 status, 为状态字节寄存器组启用使能寄存器中的位。 使能寄存器可定义事件寄存器中的哪些位将被报告给“状态字节”寄存器组。 使能寄存器是可读写的。(*SRE).

See also sre.

int CommonCommand::sre()

查询服务请求是否启用 (*SRE?).

See also setSre.

OpenVisa::StatusByteRegisters CommonCommand::stb()

读取状态字节查询。 查询状态字节寄存器组的条件寄存器并返回一个十进制值，该值等于寄存器中设置的所有位的二进制加权值总和。 条件寄存器可不间断地监控仪器的状态。 条件寄存器位实时更新；它们既不锁定亦无缓冲。

bool CommonCommand::tst()

执行仪器的基本自检并返回通过/失败指示.

void CommonCommand::wai()

配置仪器的输出缓冲区，等待所有未决操作完成之后，再通过接口执行任何其他命令。
