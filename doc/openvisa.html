enum class OpenVisa::AddressType

Visa连接地址类型.

ConstantValueDescription
OpenVisa::AddressType::RawSocket0原始Socket连接方式，必须要启用终结符和终结字符串.
OpenVisa::AddressType::SerialPort1串口通信.
OpenVisa::AddressType::USB2USBTMC通信. 注意，使用USB连接必须安装USBTMC驱动程序.如果安装NI-VISA的USBTMC驱动程序是无法访问的，应安装其他USBTMC驱动程序，参考 Zadig.
OpenVisa::AddressType::VXI113使用VXI11通信协议与设备通信.
OpenVisa::AddressType::HiSLIP4使用HiSLIP通信协议与设备通信.

enum class OpenVisa::StatusByteRegister

状态字节.

ConstantValueDescription
OpenVisa::StatusByteRegister::Unused10x0001未使用
OpenVisa::StatusByteRegister::Unused20x0002未使用
OpenVisa::StatusByteRegister::ErrorQueue0x0004一个或多个错误存在于队列中，使用SYSTem:ERRor?读取和删除错误.
OpenVisa::StatusByteRegister::QuestionableDataSummary0x0008在可疑数据寄存器中设置一个或多个位（必须启用这些位，请参见 STATus:QUEStionable:ENABle）.
OpenVisa::StatusByteRegister::MessageAvailable0x0010仪器输出缓冲区中的数据可用.
OpenVisa::StatusByteRegister::StandardEventSummary0x0020在标准事件寄存器中设置一个或多个位（必须启用这些位，请参见 *ESE）.
OpenVisa::StatusByteRegister::MasterSummary0x0040一个或多个位在状态字节寄存器中设置且可能生成服务请求 (RQS)。 必须使用 *SRE 启用位.
OpenVisa::StatusByteRegister::StandardOperationSummary0x0080在标准操作寄存器中设置一个或多个位（必须启用这些位，请参见 STATus:OPERation:ENABle）.
