Detailed Description

Note: 该类型仅用作中间件来转换，当其长期保存时，不能进行转换，只能在该值消亡时转换其中文本.

Member Function Documentation

AsciiArray::AsciiArray(std::string &&buffer, size_t offset = 0)

从 buffer 构造文本数组解析器并接管 buffer 的生命周期，跳过前面 offset 个字符串进行解析.例如：

 AsciiArray<int>(readAll());
 // or
 std::string buf = readAll();
 AsciiArray<double>(std::move(buf));

std::vector<T> AsciiArray::operator std::vector<T>() const &&

隐式转换为vector.

 std::vector<double> vec = AsciiArray<vec>(readAll());
